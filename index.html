Welcome!


My name is Nick, and I am currently a senior financial analyst at CUNA Mutual Group supporting IT. While my interest has mostly been focused around finance and data science, I always thought it would be fun to build my own website. Throughout the years I have slowly built up my knowledge of various web development languages (HTML, CSS, Javascript, etc.), and after watching some tutorials about using a Python module (Flask) as a back end and seeing some excellent examples online (a big thanks to John Yang for his walkthrough on static websites with Frozen Flask, which can be found here ), I finally had the basic tools to create my own "resume site" to showcase my skills in a different format than LinkedIn allows. I know I still have a long way to go before I can create a really cool site, but that is the fun of it, right?

Just like I mention on LinkedIn with my Kaggle profile, more work is always being added, so please check back often!
